Services

To create a website you would have to go through the following stages. We encourage you to read what creating a website is all about!

Web Design

Web design is the process of creating websites. There are several different aspects of web design, including webpage layout, and graphic design. While the terms web design and web development are often used interchangeably, web design is actually a sub-category of web development. This is a process which usually consists of using HTML and CSS while the development is more of the JavaScript and the backend languages.

Web Development

You may think web development is the same as web design; well you are strongly mistaken. The development stage of creating a website is where you create the functionality of the site. Design and development go hand in hand as without the design the website will be very plain and boring. While without the functionality of the site it will be rendered pretty much useless. This is usually the stage which takes the most time to complete.

Hosting

When you order a website from us you have two options; we develop the website and we send you the folder with the code, or we host it ourselves (monthly fee). If you are confident that you would be able to host it go ahead, but, if you don't want to get your hands dirty we are here to help! Without hosting your website will be useless as nobody will be able to see it. If you have any questions about this option don't be afraid to contact us!
